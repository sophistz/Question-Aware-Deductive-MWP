## Reproduction
To run the MAWMPS dataset:
```sh
bash ./scripts/run_mawps_five_fold.bash
```

To run the Math23k dataset:
```sh
bash ./scripts/run_math23k_train_test.bash
```

To run the MathQA dataset:
```sh
bash ./scripts/train_mathqa.bash
```

To run the Math23k dataset:
```sh
bash ./scripts/run_math23k_train_test.bash
```
